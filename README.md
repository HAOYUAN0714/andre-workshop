### 功能實現概述
```
商品搜尋功能
商品購物車
下單結帳流程
優惠券功能
(加入金額限制，並對各種狀況增加訊息回饋)
```

### 使用技術
```
Vue Cli 3
VueRouter
ESlint 的 aribnb 優化程式碼
RWD 響應式網頁設計
Bootstrap4
SCSS
API串接 ( 由六角學院提供 )
```
